## 설치 방법 (시험 버전 : 셀프 서명 인증서 포함)

1. `CctvSubmarineViewWINUI3_1.0.3.0_x64.cer`를 더블 클릭합니다.

2. **인증서 설치**를 선택합니다.

3. **로컬 컴퓨터(Local Machine)** 를 선택합니다.

4. **인증서 저장소를 직접 선택**을 선택합니다.

5. **신뢰할 수 있는 사람(Trusted People)** 저장소를 선택합니다.

6. 설치를 완료합니다.

7. `CctvSubmarineViewWINUI3_1.0.3.0_x64.msix`를 실행하여 설치합니다.
