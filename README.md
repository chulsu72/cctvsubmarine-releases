## 설치 및 업데이트 방법 

1. 오른쪽 releases 의 tags 를 클릭하고 최신 버전을 다운로드합니다.

2. `cctvSubmarine_1.0.x.0_x64_test.zip` 압축을 해제합니다.

    ※ 처음 설치 : `자체 서명 인증서 설치 방법` 참고.

3. `CctvSubmarineViewWINUI3_1.0.x.0_x64.msix`를 더블클릭하여 설치합니다.


## 자체 서명 인증서 설치 방법 ( 처음 한번만 설치 )

1. `CctvSubmarineViewWINUI3_1.0.x.0_x64.cer`를 더블 클릭합니다.

2. **인증서 설치**를 버튼을 클릭합니다.

3. **로컬 컴퓨터(Local Machine)** 를 선택합니다.

4. **인증서 저장소를 직접 선택**을 선택합니다.

5. **신뢰할 수 있는 사람(Trusted People)** 저장소를 선택합니다.

6. 인증서 설치를 완료합니다.
